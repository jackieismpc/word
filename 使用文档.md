# 单词对对碰使用文档

## Mac 使用

启动方式二选一：

1. 终端启动

```bash
cd /Users/jackie/Documents/Codex/2026-05-21/plugin-browser-openai-bundled-chrome-https
./start-word-match.sh
```

2. 双击启动

- 双击 [start-word-match.command](/Users/jackie/Documents/Codex/2026-05-21/plugin-browser-openai-bundled-chrome-https/start-word-match.command)

启动后：

- 默认地址是 `http://127.0.0.1:12345`
- 如果 `12345` 被占用，会自动换到下一个空闲端口，比如 `12346`
- 启动脚本会直接打印最终实际地址
- 如果浏览器里还停在旧页面，刷新一次即可加载最新界面

停止方式：

- 推荐在终端执行：

```bash
cd /Users/jackie/Documents/Codex/2026-05-21/plugin-browser-openai-bundled-chrome-https
./stop-word-match.sh
```

- 如果当前服务就是在前台终端里运行，也可以直接按 `Ctrl + C`
- 服务会优雅退出并释放端口

## Windows 使用

推荐方式是通过 Tailscale 使用，不需要在 Windows 上安装 Node。

步骤：

1. 先在 Mac 上启动本项目
2. Mac 和 Windows 都安装并登录 Tailscale
3. 两台设备加入同一个 Tailnet
4. 查看 Mac 启动脚本打印出来的 `Tailscale 地址`
5. 在 Windows 浏览器打开这个地址

示例：

```text
http://100.116.24.4:12345
```

注意：

- 如果默认端口 `12345` 被占用，脚本会自动换端口
- Windows 端要使用脚本最终打印出来的那个完整地址
- Windows 端直接用浏览器打开即可，不需要安装 Node 或其他开发环境
